This Power BI dashboard provides a comprehensive analysis of **hotel performance metrics** within the hospitality industry. The interactive dashboard enables stakeholders to explore **key KPIs**, monitor trends, and gain actionable insights into revenue generation, occupancy, pricing efficiency, and platform performance.

### ‚ú® Key Features
- **Overall Performance Overview:** Analyze core metrics such as Revenue, ADR (Average Daily Rate), RevPAR (Revenue per Available Room), and Occupancy %.
- **Realisation & Revenue Insights:** Track Realisation % and platform-wise performance to evaluate booking success rates.
- **Week-on-Week Comparison:** Measure performance growth or decline across key KPIs like Revenue, ADR, and Realisation %.
- **Granular Analysis:** Explore hotel performance segmented by City, Property, and Room Type.
- **Interactive Slicers:** Filter data dynamically by City, Room Type, and Property for deeper insight.
- **Trend Analysis:** Identify operational and revenue trends over time to support decision-making.

### üõ†Ô∏è Technical Details
This project was developed using **Power BI Desktop**. The data model connects multiple datasets representing hotel bookings, room types, and property details for an integrated view of performance.
- **Source Data:** Simulated hospitality dataset (Excel/CSV format) representing real-world booking and revenue patterns.
- **Data Transformation:** Power Query was used for data cleaning, column standardization, and relationship creation.
- **DAX Measures:** Custom DAX calculations were built to compute KPIs like ADR, RevPAR, Occupancy %, Realisation %, and Week-on-Week % changes.
- **Data Model:** Designed using a star schema for optimal performance and maintainability.
